# Lập trình web

Trang web bán quần áo

## Công nghệ sử dụng

**Client:** HTML, CSS, JS

**Server:** Springboot, Mysql

**Deploy:** Docker


## Cài đặt
### Cài đặt backend
#### Yêu cầu môi trường
Môi trường yêu cầu để chạy backend:

* Java 11
* mysql:5.7
* maven 3.8.6

### Cài đặt frontend

Frontend là web tĩnh nên có thể dùng các công cụ chạy file index.html lên
